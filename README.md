 # PennyKeeper | Financial Habit Tracking App

## How to Use the App

- Minimum Android version: Android 9 (Pie)
### Configure Mic
```
adb emu avd hostmicon
```
or
```
adb devices
adb -s <device shown from previous command> emu avd hostmicon
```
